Model selection on dataset2:¶

This data set contains:

  • Audio recording tags
  • Recording length
  • Mfcc 1 to 4 as each is binned to 9 section:
    • Max
    • Min
    • Median
    • Mean
    • Variance
  • Polarity of the conversation
  • Subjectivity of the conversation

All of the data was then normalized to make the model section a fair game between the tree and non tree base algorithms. Without normalization process Logistic Regression and K-Neighbors Classifier will not be able to make any accurate prediction.

At then end we have 587 predictors.

  1. Reading the dataset
  2. Normalizing
  3. Converting recording tags to dummy variables
  4. Makeing dataset
  5. Model selection
    • Random Forest Classifier
    • Gradient Boosting Classifier
    • Logistic Regression
    • K-nearest Neighbors
  6. Feature importance
